# The Architect of Optimism and the Disciple of the Deal

An IKnot Report by Rashad F.

This project is a static editorial landing page adapted from a research report on Norman Vincent Peale's ideological influence on Donald Trump's religious and political worldview.

## Files

- `index.html` - the full landing page, including layout, styling, interactive tabs, citations, and report sections.
- `Peale to Trump .docx` - supporting source material used to expand the Trump-Peale connection section.
- `The Architect of Optimism and the Disciple of the Deal_ Norman Vincent Peale’s Ideological Impact on Donald Trump’s Religious and Political Worldview.md` - original report draft.

## Viewing

Open `index.html` in a browser. No build step or local server is required.
